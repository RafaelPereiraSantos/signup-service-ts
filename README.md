## Signup-Service

A small mpv containing endpoints using [NextJs](https://nextjs.org/) and a worker in [rabbitMq](https://github.com/rabbitmq/rabbitmq-tutorials/blob/master/javascript-nodejs)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
